O Git Bash é um terminal extendido para otimizar o uso do Git.

Criar novo repositório:
git init

Verificar estado dos arquivos/diretórios:
git status

Adicionar arquivos solicitados ao ambiente de stage:
git add

Fazer commit:
commit -m “atualização”

Subir modificações no repositório remoto:
git push


PASSO A PASSO CRIAR REPOSITORIO E REALIZAR ATUALIZAÇÕES:
1 - git clone git@github.com:luan...(abrir pasta antes pele git bash)
2 - cd "pasta repositorio"
3 - git status
4 - git add .
5 - git status
6 - git commit -m "Inclusão ..."
7 - git status
8 - git push origin main
